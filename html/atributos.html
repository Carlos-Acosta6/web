Atributos Globales

¿Qué son?

Los atributos globales son aquellos que se pueden aplicar a casi cualquier etiqueta HTML, sin importar su tipo o función. Sirven para controlar aspectos como la identificación, el estilo o el comportamiento de los elementos. Estos atributos permiten personalizar y manipular los componentes del documento de forma más precisa y flexible, tanto con CSS como con JavaScript.


id

El atributo id se usa para asignar un identificador único a un elemento HTML. Este sirve para referirse a ese elemento desde CSS o JavaScript. Por ejemplo: (p id="texto1">Hola -p). No puede repetirse dentro del mismo documento.





class

El atributo class permite agrupar varios elementos bajo una misma categoría. Es muy útil para aplicar estilos con CSS o realizar acciones con JavaScript. A diferencia de id, se puede usar en muchos elementos a la vez.





style

El atributo style se utiliza para aplicar estilos directamente dentro de la etiqueta HTML. Permite modificar colores, tamaños o fuentes de forma rápida, aunque lo más recomendable es usar archivos CSS externos.






Atributos Form

¿Qué son?

Los atributos de formulario controlan el comportamiento de las etiquetas que forman parte de un form. Ayudan a definir cómo se envían los datos, a qué dirección van y qué método se utiliza. Son esenciales para crear formularios funcionales y seguros en una página web.


action

El atributo action indica la dirección (URL) a donde se enviarán los datos del formulario una vez que el usuario lo envíe. Por ejemplo: form action="procesar.php">.




method

El atributo method especifica cómo se enviarán los datos del formulario. Los valores más comunes son GET (los datos se ven en la URL) y POST (los datos se envían de forma oculta, más segura).





Atributos Input

¿Qué son?

Los atributos de input determinan el tipo y el comportamiento de los campos de entrada dentro de un formulario. Permiten definir valores predeterminados, límites o restricciones, mejorando la experiencia del usuario y la validación de datos.


type

El atributo type define el tipo de dato que se quiere recibir, como texto, contraseña, número, correo, fecha, entre otros. Ejemplo: input type="email".





placeholder

El atributo placeholder muestra un texto dentro del campo antes de que el usuario escriba. Sirve como guía o ejemplo del tipo de información que debe introducir.





value

El atributo value establece un valor inicial o predeterminado para el campo. También se usa para mostrar el texto en los botones de formulario, como en input type="submit" value="Enviar".
