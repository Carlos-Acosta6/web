Atributos Globales

¿Qué son?

Los atributos globales son aquellos que se pueden aplicar a casi cualquier etiqueta HTML, sin importar su tipo o función. Sirven para controlar aspectos como la identificación, el estilo o el comportamiento de los elementos. Estos atributos permiten personalizar y manipular los componentes del documento de forma más precisa y flexible, tanto con CSS como con JavaScript.


id

El atributo id se usa para asignar un identificador único a un elemento HTML. Este sirve para referirse a ese elemento desde CSS o JavaScript. Por ejemplo: (p id="texto1">Hola -p). No puede repetirse dentro del mismo documento.





class

El atributo class permite agrupar varios elementos bajo una misma categoría. Es muy útil para aplicar estilos con CSS o realizar acciones con JavaScript. A diferencia de id, se puede usar en muchos elementos a la vez.





style

El atributo style se utiliza para aplicar estilos directamente dentro de la etiqueta HTML. Permite modificar colores, tamaños o fuentes de forma rápida, aunque lo más recomendable es usar archivos CSS externos.






Atributos Form

¿Qué son?

Los atributos de formulario controlan el comportamiento de las etiquetas que forman parte de un form. Ayudan a definir cómo se envían los datos, a qué dirección van y qué método se utiliza. Son esenciales para crear formularios funcionales y seguros en una página web.


action

El atributo action indica la dirección (URL) a donde se enviarán los datos del formulario una vez que el usuario lo envíe. Por ejemplo: form action="procesar.php">.




method

El atributo method especifica cómo se enviarán los datos del formulario. Los valores más comunes son GET (los datos se ven en la URL) y POST (los datos se envían de forma oculta, más segura).





Etiquetas de Agrupacion

¿Qué son?

Estas etiquetas se utilizan para organizar y agrupar bloques de contenido dentro del cuerpo del documento. Ayudan a estructurar el texto y a mantener el orden visual y semántico de la página.


p

La etiqueta p define un párrafo. Es una de las más comunes y sirve para separar bloques de texto con un espacio entre ellos.





hr

La etiqueta hr crea una línea horizontal que divide secciones o temas dentro de una página. Es útil para separar contenido visualmente.





ul

La etiqueta ul se usa para crear listas no ordenadas, es decir, con viñetas o puntos. Dentro de ella se colocan los elementos con li.





div

La etiqueta div sirve para agrupar varios elementos en un solo bloque. Es muy utilizada para maquetar páginas y aplicar estilos específicos a ciertas secciones.





figure

La etiqueta figure se usa para incluir contenido visual como imágenes, diagramas o gráficos, junto con su descripción o pie de imagen.





figcaption

La etiqueta figcaption añade una descripción o título a un elemento . Ayuda a dar contexto a las imágenes o ilustraciones.





Etiquetas textuales

¿Qué son?

Estas etiquetas afectan directamente al texto, modificando su estilo o función dentro de un documento HTML. Permiten crear enlaces, aplicar formatos o resaltar partes del contenido.


a

La etiqueta a crea enlaces a otras páginas o secciones. Es fundamental para la navegación web y puede incluir tanto direcciones internas como externas.





span

La etiqueta span se usa para aplicar estilos o scripts a una parte específica del texto sin romper la línea. Es muy útil para resaltar palabras o frases.





sub

La etiqueta sub muestra texto en subíndice, colocándolo ligeramente por debajo de la línea normal. Se usa en fórmulas químicas o matemáticas.





sup

La etiqueta sup muestra texto en superíndice, es decir, más pequeño y elevado. Se usa para exponentes o notas al pie.





cite

La etiqueta cite se utiliza para indicar el título de una obra, libro o fuente. Suele mostrarse en cursiva para destacar su referencia.






Etiquetas Semanticas

¿Qué son?

Las etiquetas semánticas describen claramente el propósito de cada parte del contenido. Facilitan la lectura tanto para los usuarios como para los motores de búsqueda, mejorando la estructura y accesibilidad de la web.



nav

La etiqueta nav agrupa los enlaces principales de navegación del sitio. Generalmente se coloca al inicio o en menús laterales.





section

La etiqueta section define una sección o bloque de contenido con un tema específico. Se usa para dividir la página en partes organizadas.





article

La etiqueta article contiene información independiente, como una noticia, entrada de blog o publicación. Es un bloque que puede existir por sí mismo.





header

La etiqueta se usa para la parte superior de una página o sección. Suele incluir el logo, el título o el menú de navegación.





main

La etiqueta main identifica el contenido principal de una página, es decir, la información más importante o central que el usuario busca.





footer

La etiqueta footer se encuentra al final de la página o de una sección. Suele contener información de contacto, derechos de autor o enlaces secundarios.
