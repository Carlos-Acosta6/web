Acerca del Proyecto

Este sitio web fue creado con el propósito de enseñar los fundamentos del desarrollo web, utilizando los tres pilares principales: HTML, CSS y JavaScript. El contenido explica cómo se estructura una página con HTML, cómo se aplican estilos visuales con CSS y cómo se agrega interactividad mediante JavaScript. En la sección de HTML se abordan las etiquetas más importantes, sus atributos y la manera correcta de organizar un documento. En CSS, se detallan los tipos de hojas de estilo (en línea, internas y externas) y su aplicación práctica para mejorar la apariencia del sitio. Finalmente, JavaScript introduce la lógica y funcionalidad dinámica para enriquecer la experiencia del usuario. Este proyecto busca servir como una introducción completa al desarrollo web, con explicaciones simples y ejemplos que ayuden a los principiantes a comprender cómo se combinan estas tres tecnologías en la creación de páginas modernas.
