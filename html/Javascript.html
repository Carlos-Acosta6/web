¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. A diferencia de HTML y CSS, que solo estructuran y dan estilo, JavaScript puede reaccionar a acciones del usuario, validar formularios, mostrar mensajes o cambiar el contenido sin recargar la página. Es uno de los lenguajes más usados en el desarrollo web y funciona directamente en el navegador.
