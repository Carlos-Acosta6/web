Etiquetas de documento

¿Qué son?

Las etiquetas de documento son las que forman la estructura básica de cualquier archivo HTML. Indican el inicio y el final del documento, además de dividirlo en secciones importantes como el encabezado y el cuerpo. Son las primeras etiquetas que debe aprender cualquier persona que empiece a crear páginas web. Sin ellas, el navegador no podría interpretar correctamente el contenido.


HTML

La etiqueta html marca el inicio y el fin de un documento HTML. Todo el contenido de la página, incluyendo el encabezado y el cuerpo, debe ir dentro de ella. Indica al navegador que el texto dentro pertenece a una página web estructurada con HTML.





head

La etiqueta head contiene información que no se muestra directamente en la página, pero que es esencial para su funcionamiento. Aquí se colocan el título, los metadatos, enlaces a hojas de estilo y scripts. Es el “cerebro” del documento HTML.





body

La etiqueta body encierra todo el contenido visible de una página web. Dentro se colocan los textos, imágenes, enlaces, listas y cualquier elemento que el usuario pueda ver o interactuar con. Es el “cuerpo” del documento HTML.






Etiqueta de meta datos

¿Qué son?

Estas etiquetas proporcionan información sobre la página que no se ve directamente, pero que ayuda a los navegadores, buscadores y redes sociales a interpretarla mejor. Se colocan dentro del head y son esenciales para la optimización web y el diseño.


Meta

La etiqueta meta define metadatos como la codificación de caracteres, la descripción del sitio o las palabras clave. Ayuda al navegador y a los motores de búsqueda a entender de qué trata la página.





Style

Con la etiqueta style se pueden incluir reglas CSS directamente en el documento HTML. Sirve para dar color, tamaño o formato al contenido sin necesidad de un archivo externo.





Title

La etiqueta title establece el título que aparece en la pestaña del navegador. También es importante para el SEO, ya que los buscadores lo usan para identificar la página.





Base

La etiqueta base define una URL base para todos los enlaces y rutas de la página. Esto permite escribir direcciones relativas en lugar de absolutas, facilitando la organización del sitio





Link

La etiqueta link conecta el documento con archivos externos, como hojas de estilo CSS o íconos del sitio. Se utiliza mucho para enlazar estilos que mejoran el diseño visual.





Etiquetas de Agrupacion

¿Qué son?

Estas etiquetas se utilizan para organizar y agrupar bloques de contenido dentro del cuerpo del documento. Ayudan a estructurar el texto y a mantener el orden visual y semántico de la página.


p

La etiqueta p define un párrafo. Es una de las más comunes y sirve para separar bloques de texto con un espacio entre ellos.





hr

La etiqueta hr crea una línea horizontal que divide secciones o temas dentro de una página. Es útil para separar contenido visualmente.





ul

La etiqueta ul se usa para crear listas no ordenadas, es decir, con viñetas o puntos. Dentro de ella se colocan los elementos con li.





div

La etiqueta div sirve para agrupar varios elementos en un solo bloque. Es muy utilizada para maquetar páginas y aplicar estilos específicos a ciertas secciones.





figure

La etiqueta figure se usa para incluir contenido visual como imágenes, diagramas o gráficos, junto con su descripción o pie de imagen.





figcaption

La etiqueta figcaption añade una descripción o título a un elemento . Ayuda a dar contexto a las imágenes o ilustraciones.
