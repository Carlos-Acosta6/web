¿Qué es HTML?

El documento que lee el navegador está escrito en un lenguaje de marcado llamado HTML,
que son las siglas de HyperText Markup Language (Lenguaje de marcas de hipertexto),
o lo que es lo mismo, un lenguaje de etiquetas que permite incluir o hacer referencia a todo tipo de información.

Historia HTML

HTML fue creado por Tim Berners-Lee en 1991 para estructurar y enlazar documentos en la web.
En 1995 se lanzó HTML 2.0, la primera versión estándar. Con HTML 4.01 (1999) se mejoró el diseño y se separó el contenido del estilo.
Después surgió XHTML (2000), una versión más estricta basada en XML. Finalmente, HTML5 (2014) modernizó el lenguaje con soporte para video, audio, gráficos y
etiquetas semánticas. Hoy en día, HTML sigue siendo la base de todas las páginas web.

Editores de HTML

Los editores de HTML son programas donde se escriben y editan los códigos de una página web. Existen editores simples como Bloc de notas o Notepad++, y otros más avanzados como Visual Studio Code, Sublime Text o Atom. Estos ofrecen funciones útiles como coloreado de sintaxis, autocompletado y vista previa, lo que facilita mucho la creación de páginas web.

En el se coloca el título y la información del documento, mientras que en el va el contenido visible como textos, imágenes o enlaces. Con solo unas pocas líneas de código, ya se puede crear una página básica.

¿Qué son Etiquetas en HTML?

Las etiquetas en HTML son instrucciones que indican cómo se debe mostrar o estructurar un contenido. Siempre van entre los signos <> y suelen tener una etiqueta de apertura y otra de cierre, como p. Cada etiqueta cumple una función específica, por ejemplo h1 para títulos, p para párrafos o imgpara imágenes. Son esenciales para que el navegador entienda la estructura del documento.

¿Qué son Atributos en HTML?

Los atributos son propiedades que se agregan dentro de una etiqueta para darle más información o modificar su comportamiento. Por ejemplo, en img src="foto.jpg" alt="Foto", los atributos src y alt indican la fuente de la imagen y su descripción. Todos los atributos se colocan dentro de la etiqueta de apertura y ayudan a personalizar cómo se muestra o funciona cada elemento.

Reglas fundamentales en HTML.

En HTML, cada documento debe tener una estructura básica: !DOCTYPE html, html, head y body. Las etiquetas deben abrirse y cerrarse correctamente para evitar errores. Además, no se debe olvidar la sangría y el orden, ya que ayudan a mantener el código limpio y entendible. Por último, es importante respetar las mayúsculas y minúsculas en las rutas o nombres de archivos.
